## 乾卦

**卦辞**

元，亨，利，贞

元：准备好，一个事物开始前一定要准备好，即谋定而后动。

亨：互通有无，在慎始的前提下就可以通达(做事顺利)了

利：做事情后的获利(金钱，名利)

贞：公正即做事情守正道，不是走歪门邪道。比如搞诈骗牟利，上面三步都按照规则走也骗到钱了，但最终因为不守贞而抓走。

所以做事情时动机要纯正，处事要谨慎，利益要合理，为人要公正。这四个步骤有点像`PDCA`，不是所有阶段走一遍就结束了，而是螺旋上升的过程

**卦象**

*天行健，君子以自强不息*

天空**长久的**进行云行雨展,这样空气才会流通草木才会生长。君子要学习天空的这样的品质，自强不息。

**初九**

*潜龙勿用* 

以人为例，当一个人来到一个新的地方不应该是多多的发表意见和建议，而是聆听他人的看法，了解周围刚刚解除的人和事，所以即使存在潜力也要“勿用”。当了解了事物之后才开始发挥自己的作用和能力。

> 包括自己是某个行业的专家，到了新的地方也要先了解，因为新的地方会有非常特殊的做法，一定要先了解

潜龙勿用告诉人们不要一味的求表现。不过不要一直潜在那边，该你表现的时候一定要表现。这样就到了二爻

**九二**

*见龙在田，利见大人*

度过了潜龙阶段，人开始表现。但人一旦开始表现会遭受四面八方的打击，而且表现的越好越遭人妒忌。

**九三**

*君子终日乾乾，夕惕若；厉，无咎*

一旦表现的过好就达到了这个阶段，而这个阶段就需要多加小心，因为你处于聚光灯之下所有人都可以研究你而你都不知道攻击来自何处，所以处于这个阶段需要有**忧患意识**。

> 一个人没有名，没有钱自我报废的情况很少。人一旦有名有钱有势就需要警惕了，一旦尾巴翘起来的时候就快自我报废了。这个阶段是需要高度警惕的，俗话说要爱惜自己的羽毛。

人还要有远见，不能做事情有后遗症，给自己挖坑。这就是无咎

**九四**

*或跃在渊，无咎*

跳跃阶段，要么更上一层楼，要么跌入谷底。但高处不胜寒，越往上越孤单。所以不需要盲目向上走。

> 如果像要你向上跃的人不是为了你自己而是为了他们的利益。捧你的人类似抬轿子的，你相当于坐轿子的。你去哪里完全由抬轿子的决定。

**九五**

*飞龙在天，利见大人*

人生最高境界，自由自在。但要小心水能载舟，亦能覆舟。所以要关心下面的人，度量要大

**上九**

*亢龙有悔*

物极必反，就像物体上升到顶点后势必快速的下坠。

可以类比为断了线的风筝虽然飞的很高但失去了控制，而且早晚会掉下来。

**附**

外国人讲能力，中国人讲本事。而本事就是既有能力又很受人们欢迎。

人生最难的就是知道自己的分在哪里，也就是找到那个度。俗话说不要过度
